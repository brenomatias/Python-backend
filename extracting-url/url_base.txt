    Fatiamento de string
        O fatiamento é uma funcionalidade do Python onde você consegue extrair um pedaço de uma string

        O fatiamento você passa duas posições para a sua string original e essas duas posições serão as posições de início e de fim de onde você quer extrair ex [0:2]

         o fatiamento tem uma característica onde o primeiro argumento dele, no nosso caso foi o 0, é inclusivo, mas o segundo argumento, o argumento final, é exclusivo, ou seja, ele não vai incluir o caractere naquela posição

         O operador de fatiamento [a:b], utilizado para obter uma substring desde o índice a até o índice b - 1 da string original. Lembrando que b - 1 pois o segundo argumento do fatiamento é exclusivo.

    .find() -> retorna o index da posição da substring
        caso a substring nao exista, retorna -1
